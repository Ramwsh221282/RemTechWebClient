Инструкция по настройке Google почтового отправителя:
Шаг 1. Создайте почтовый ящик на сервисе: https://mail.google.com - это будет почтой отправителя.
Шаг 2. Создайте ключ для внешего приложения: https://myaccount.google.com/apppasswords - Это будет SMTP ключом отправителя.
Шаг 3. Оформите форму добавления почтового отправителя на странице, используя почту и ключ.
