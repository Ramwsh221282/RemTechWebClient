Инструкция по настройке Yandex почтового отправителя:
Шаг 1. Создайте почтовый ящик на сервисе: http://mail.yandex.ru - это будет почтой отправителя.
Шаг 2. Создайте IMAP, POP3, SMTP ключ для внешего приложения: https://id.yandex.ru/security/app-passwords - Это будет SMTP ключом отправителя.
Шаг 3. Оформите форму добавления почтового отправителя на странице, используя почту и ключ.
