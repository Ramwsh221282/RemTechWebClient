Инструкция по настройке Mail Ru почтового отправителя:
Шаг 1. Создайте почтовый ящик на сервисе: https://mail.ru - это будет почтой отправителя.
Шаг 2. Создайте пароль для внешего приложения: https://account.mail.ru/user/2-step-auth/passwords - Это будет SMTP ключом отправителя.
Шаг 3. Оформите форму добавления почтового отправителя на странице, используя почту и ключ.
